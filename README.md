# Rasa Nusantara - Landing Page

Landing page responsif dan modern untuk bisnis kuliner **Rasa Nusantara**. Dibangun menggunakan HTML5 murni dan utility-first CSS framework **Tailwind CSS**.

## Fitur Utama
- 🌟 **Desain Estetik & Modern:** Menggunakan palet warna hangat bertema rempah tradisional.
- 📱 **Responsif:** Tampilan optimal di perangkat mobile, tablet, maupun desktop.
- ⚡ **Ringan:** Memanfaatkan CDN Tailwind CSS untuk performa loading yang cepat.
- 🗺️ **Seksi Navigasi Lancar:** Fitur smooth scroll antarbagian halaman.

## Cara Menjalankan Secara Lokal
1. Clone repositori ini atau unduh berkas zip-nya.
2. Buka berkas `index.html` langsung di browser Anda (Google Chrome, Edge, Safari, dll.).
